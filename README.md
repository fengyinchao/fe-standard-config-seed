# standard-template

这是一个集多数常用配置于一身的前端项目模版，包括

- typescript
- eslint
- stylelint
- commitlint
- prettier
- husky
- changelog
- editorconfig
- standversion
- license

## How To Use

```shell
# step1:clone
git clone https://github.com/fengyinchao/standard-template.git
# step2:cover .git
cd standard-template && git init
# step3:install
npm i
# step4:add your code
//...
git add .
# step5:commit
git cz
# step6:generate changelog
npm run release
```
